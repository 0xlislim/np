## Step 1: Create Profile README Repository

### 1.1 Create the repository
1. Go to https://github.com/new
2. Repository name: `0xlislim` (must match your username exactly)
3. Set to **Public**
4. Check "Add a README file"

2. Click the pencil icon (Edit)
